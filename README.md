# Task Manager CLI

A simple C# console application for managing your tasks with persistent JSON storage.

## Features
- Add, view, delete, and mark tasks as complete
- Automatic save to `tasks.json`
- Clean menu-driven interface

## Requirements
- .NET 6.0+

## Usage
```bash
dotnet run
```

## Menu Options
1. Add Task - Create new task with title and details
2. View Tasks - See all tasks with status
3. Delete Task - Remove a task
4. Mark Complete - Toggle completion status
5. Exit

## Data Storage
Tasks saved automatically to `tasks.json` in app directory.
